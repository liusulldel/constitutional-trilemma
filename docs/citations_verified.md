# Reference Notes

This file keeps citation cautions for the working paper. It is intentionally
plain and conservative so the public repo does not read like an internal audit
log.

## Notes To Preserve

- Birrell and Pass (2011) is a related approximate strategy-proofness result, but
  it is not the same theorem as the Mossel-Racz quantitative
  Gibbard-Satterthwaite result. Do not use the Birrell-Pass expression as the
  strategic leg's bound.
- Isaksson, Kindler, and Mossel (2012) appeared in *Combinatorica*, not *Annals
  of Mathematics*.
- Mossel and Racz (2015) appeared in *Combinatorica* and is the relevant
  quantitative Gibbard-Satterthwaite reference.
- Cover and Thomas should be cited for cross-entropy, KL decomposition, and data
  processing in the appropriate sections, not as a single catch-all theorem.
- Ge et al. (2024) should be cited with the verified title and author list:
  "Axioms for AI Alignment from Human Feedback."
